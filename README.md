# C-Grid-Based-Simple-Database
A lightweight, coordinate-based data management system written in C++. This project uses 2D matrix logic to store strings and provides persistent storage through file I/O operations.
## ✨ Key Features
* **Coordinate-Based Storage:** Store and retrieve data using `[X][Y]` grid coordinates.
* **Full CRUD Operations:** Support for adding, searching, listing, and deleting entries.
* **Persistent Storage:** Automatically saves data to `database.txt` for future sessions.
* **Interactive CLI:** A simple, menu-driven interface for easy navigation.

## 🛠️ Technical Stack
* **Language:** C++
* **Data Structures:** 2D String Array & Boolean Masking
* **File I/O:** `fstream` library for data persistence
